# Проект выставки депо детских воспоминаний

## Какие ресурсы и ИИ использовались и для чего?

### WebStorm

IDE для работы с проектом.

### Claude

Программирование анимации, помощь с логикой работы js функций, помощь с компонентами.

### Cursor

Верстка блоков, простых страниц, подключение шрифтов, написание базовых стилей. 

### Stackoverflow

Поиск ответов на вопросы по реализации анимации через js, структуры
